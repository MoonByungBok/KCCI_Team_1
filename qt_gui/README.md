### 2. 유즈케이스 시나리오

##### 목표 : CCTV 로 칼 든 사람을 검출해서 주변에 알림
```
1. CCTV를 켠다
2. CCTV로 사람을 찾는다.
3. 찾은 사람이 칼을 들고 있는지 체크한다.
4. 칼을 들고 있으면 서버에 GPS 데이터와 캡쳐 이미지를 전송한다.
5. 사이렌을 울린다.
```
